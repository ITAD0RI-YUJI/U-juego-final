### Controles del juego
* __Mover:__
Para poder mover al avatar usa las flechas del teclado: 
    - (←) Izquierda
    - (→) Derecha
    - (↑) Saltar
<br>

* __Armas:__
El avatar cuenta con un total de dos armas:
    - (A) Disparar
    - (W) Corte con catana
